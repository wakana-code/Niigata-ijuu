新潟ってどんな都市？住んでみてどんな感じ？地方移住を検討したい！そんな人のための新潟特化の移住先を見つけられるアプリです！新潟の街が特徴やタグ別に整理されて一覧で見ることができ、さらに診断することで希望の条件に合った町をおすすめ順に見ることができます。さらにミニゲームで信濃川を川下りしながら新潟の街について学ぶこともできます！新潟の地域創生のため、移住を促進するアプリを制作しました。
